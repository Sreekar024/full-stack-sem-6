## ✨ Features Implemented

### ✅ Login Page
- Username & Password input
- Form validation
- POST /login API integration
- JWT token storage in **sessionStorage**
- Error messages
- Loading states
- Redirect to dashboard on success

### ✅ Protected Dashboard
- Session verification
- Auto-redirect if no token
- GET /protected API call
- Bearer token in Authorization header
- Display protected data
- Show logged-in username
- Token info display

### ✅ Logout Functionality
- Clear token from sessionStorage
- Clear username from sessionStorage
- Redirect to login page
- One-click logout button

### ✅ Session-Based Security
- Token stored per **session** (sessionStorage, not localStorage)
- Token persists on page refresh
- Session cleared when browser tab closes
- Auto-redirect on unauthorized (401)
- Route protection
- Error handling

### ✅ Beautiful UI
- Modern gradient background (purple to pink)
- Smooth animations
- Responsive design (desktop, tablet, mobile)
- Loading spinners
- Error messages
- Success feedback
- Professional styling
